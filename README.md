# 🔎 Buscador Avanzado de Google con Flet (Python)

Aplicación de escritorio desarrollada en **Python** utilizando **Flet**, diseñada para facilitar la realización de búsquedas avanzadas en Google de manera rápida, intuitiva y eficiente.

## 🚀 Descripción

Esta aplicación permite a los usuarios construir consultas avanzadas de Google sin necesidad de memorizar operadores complejos. A través de una interfaz gráfica amigable, podés personalizar distintos parámetros de búsqueda para obtener resultados mucho más precisos.

Ideal para investigadores, estudiantes, desarrolladores… o cualquier persona cansada de pelearse con Google 😄

## ✨ Características

* 🔍 **Búsqueda avanzada simplificada**

  * Construcción automática de queries complejas de Google.

* 🌐 **Selección de fuente**

  * Filtrá resultados por sitios específicos (ej: `site:edu`, `site:gov`, etc.).

* 📄 **Filtrado por tipo de archivo**

  * Buscá archivos como:

    * PDF
    * DOCX
    * PPT
    * TXT
    * y más

* ⚙️ **Opciones adicionales**

  * Inclusión/exclusión de palabras clave
  * Búsqueda exacta con comillas
  * Operadores avanzados integrados
  * Entradas de texto para colocar tu propia fuente que no se encuentré en las opciones

* 🖥️ **Interfaz gráfica moderna**

  * Construida con **Flet**, ofreciendo una experiencia fluida y multiplataforma.

## 🛠️ Tecnologías utilizadas

* **Python 3**
* **Flet** (framework para apps UI en Python)
